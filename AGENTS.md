# Repository Guidelines (YandexAgetn)

Этот репозиторий является самостоятельным проектом. Исторически он начинался как импорт snapshot-кода OpenClaw, но git-связи с апстримом нет. Атрибуция: `ATTRIBUTION.md`.

## Быстрые команды

- Install deps: `pnpm install`
- Dev (gateway): `pnpm dev`
- UI dev: `pnpm ui:dev`
- Checks: `pnpm check`
- Tests: `pnpm test`

## Правила для изменений

- Не коммить секреты: `.env`, токены, приватные ключи, реальные номера телефонов/чатов.
- По возможности держать изменения небольшими и проверяемыми: тесты/линт перед push.
- Если правишь UI и gateway вместе, старайся добавлять хотя бы один тест на критичную логику.
