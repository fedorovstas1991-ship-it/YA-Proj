# YA-Proj

Рабочий репозиторий проекта YandexAgetn (dev-ready).

## Быстрый старт

```bash
pnpm install
pnpm build
./yagent-onboard-ui.command
```

## Разработка

- Основная ветка: `main` (только через PR)
- Для работы создавайте feature-ветки от `main`
- Перед PR запускайте локальные проверки:

```bash
pnpm build
pnpm test
```

## Где вести backlog

- Product feedback: `docs/backlog/feedback.md`
- Bugs: `docs/backlog/bugs.md`
- Fixes / changelog фиксов: `docs/backlog/fixes.md`
- Для задач на GitHub используйте шаблоны Issue в `.github/ISSUE_TEMPLATE/`
